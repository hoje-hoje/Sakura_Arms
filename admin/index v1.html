관리자 사이트 연결 설정

토큰 만드는 법 (fine-grained token, 이 저장소만 접근 가능하게)
1. github.com 로그인 → 오른쪽 위 프로필 사진 → Settings
2. 왼쪽 맨 아래 Developer settings → Personal access tokens → Fine-grained tokens
3. Generate new token 클릭
4. Repository access → "Only select repositories" → 이 저장소 선택
5. Permissions → Repository permissions → Contents: Read and write로 설정
6. Generate token → 나오는 토큰 값 복사해서 위에 붙여넣기 (한 번만 보여주니 꼭 복사)

이 값들은 이 브라우저(이 컴퓨터)에만 저장되고 GitHub 서버로만 전송됩니다. 다른 사람 컴퓨터에서는 각자 자기 이름/토큰으로 다시 설정해야 합니다.

내 이름은 저장할 때 커밋 메시지 앞에 자동으로 붙어서, 누가 언제 뭘 고쳤는지 옆 파일의 "히스토리(시계 아이콘)" 버튼에서 바로 볼 수 있게 해줍니다.
